# Интернет-магазин с выводом текущих заказов

Простой веб-проект интернет-магазина на Flask с функционалом отображения текущих заказов на главной странице. Проект демонстрирует базовую работу с базой данных заказов и вывод их в удобном виде с красивым UI.

---

## Описание

Этот проект реализует:

- Веб-сервер на Flask.
- Получение списка заказов из базы данных (через SQLAlchemy или SQL-запросы).
- Отображение заказов на главной странице в виде таблицы с улучшенным дизайном.
- Адаптивный и удобочитаемый интерфейс с использованием CSS.
